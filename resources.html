How to add a link

To add a link, just select the text you want to make a link.

How to hide Full Page Menu in order to see the content behind
How to add a Link: Just select the text and then select "make a link" then add to the link the "link" class
How to add a "fade out" effect to a Link: Just select the button or link and add the action "Fade out body on click"

important note

In Demo 2, in the Gallery Section; I have added 2 Title Boxes (where it says "Your Title Here") I did this because on mobile devices there is no "hover"action so when you go to the mobile site you would not see the "Your Title Here" because is not activated. So... you need to add the title 2 times. Just go to the landscape mode of this section and change de title as well on desktop mode. That is all. Thanks.
